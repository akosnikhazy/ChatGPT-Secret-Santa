**Prompt 1:** Write a secret santa script in javascript and HTML. It should accept names as input and randomly pair them. Every one should be paired to only one person

**ChatGPT:** Here is a simple secret santa script in JavaScript and HTML that accepts names as input and pairs them randomly:

[prompt1.html](prompt1.html)

```
<!DOCTYPE html>
<html>
<head>
  <title>Secret Santa Generator</title>
</head>
<body>
  <h1>Secret Santa Generator</h1>
  <form>
    <label for="names">Enter the names, separated by commas:</label><br>
    <input type="text" id="names"><br><br>
    <button type="button" onclick="generatePairs()">Generate Pairs</button>
  </form>
  <br>
  <h2>Pairs:</h2>
  <ul id="pairs"></ul>
  <script>
    function generatePairs() {
      // get the names from the input field
      var names = document.getElementById('names').value;
      // split the names into an array
      names = names.split(',');
      // shuffle the array of names
      names = shuffle(names);
      // clear any existing pairs from the list
      document.getElementById('pairs').innerHTML = '';
      // generate the pairs and add them to the list
      for (var i = 0; i < names.length - 1; i++) {
        var li = document.createElement('li');
        li.innerHTML = names[i] + ' is paired with ' + names[i + 1];
        document.getElementById('pairs').appendChild(li);
      }
      // add the last pair
      var li = document.createElement('li');
      li.innerHTML = names[names.length - 1] + ' is paired with ' + names[0];
      document.getElementById('pairs').appendChild(li);
    }
    // shuffle an array in-place using the Fisher-Yates shuffle algorithm
    function shuffle(array) {
      for (var i = array.length - 1; i > 0; i--) {
        var j = Math.floor(Math.random() * (i + 1));
        var temp = array[i];
        array[i] = array[j];
        array[j] = temp;
      }
      return array;
    }
  </script>
</body>
</html>

```

The script uses the Fisher-Yates shuffle algorithm to randomly shuffle the array of names, and then it generates the pairs by iterating over the shuffled array and pairing each element with the next one, with the last element being paired with the first element. The pairs are then added to an HTML list on the page.

The user can enter the names in the input field, separated by commas, and then click the "Generate Pairs" button to generate the pairs and display them on the page. The pairs are generated every time the button is clicked, so the user can keep generating new pairs until they are satisfied with the results.
